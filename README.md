# FitTrack

Weil ich keine vernünftige Offline-App gefunden habe, um meine Workouts zu tracken, habe ich diese App selbst entwickelt. Und weil mir irgendwann langweilig wurde, hab ich noch einen Lebensmitteltracker dazu gebaut — der funktioniert allerdings nur online.

## Features

- Workouts erstellen, tracken und verwalten — vollständig offline
- Trainingshistorie einsehen
- Lebensmitteltracker (benötigt Internetverbindung)

## Usage

Open `index.html` in any modern browser — no build step required.
